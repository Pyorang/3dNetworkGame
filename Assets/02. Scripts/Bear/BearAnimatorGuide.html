Detect · Attack · Hit — 이 3개는 반드시 정확히 일치

나머지(Idle, Patrol, Chase, Return)는 MoveSpeed와 IsChasing으로 블렌딩하므로 Blend Tree로 통합 가능합니다.

🔀 전이(Transition) 상세 설정 Unity에서 설정

🛠️ Unity 설정 순서
1

Parameters 탭에서 파라미터 6개 추가

Animator 창 좌측 Parameters 탭 → + 버튼
MoveSpeed(Float) · Detect(Trigger) · Attack(Trigger) · Hit(Trigger) · Die(Trigger) · IsChasing(Bool)

2

스테이트 생성

Base Layer에 우클릭 → Create State → Empty:
Idle · Patrol · Detect · Chase · Attack · Hit · Dead
Idle를 우클릭 → Set as Layer Default State
각 스테이트 Motion에 해당 애니메이션 클립 드래그

3

Blend Tree 사용 (선택)

Idle/Walk/Run을 Blend Tree로 묶으면 자연스럽습니다.
Blend Parameter: MoveSpeed
Threshold: 0=Idle, 2=Walk, 4=Run

4

Transition 연결

위 전이 표 참고, 스테이트 우클릭 → Make Transition → 대상 클릭
화살표 선택 → Inspector에서 Conditions 추가

5

AnyState → Hit, Dead

AnyState 우클릭 → Make Transition → Hit, Dead로 연결
중요: 둘 다 Can Transition To Self 체크 해제!

6

Transition Settings 세부 조정

Transition Duration: 전투 전이는 0.1초로 짧게
Has Exit Time: Detect, Attack, Hit ✅ / 나머지 ❌

7

Attack 클립에 Animation Event 추가

Attack 클립 → Animation 창 → 데미지 프레임에 Add Event
Function: OnAttackHit
